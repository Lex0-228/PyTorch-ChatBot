Бот побудований на базі машинного навчання за допомогою бібліотеки PyTorch: у файлі json_try#1.json записані всі можливі запити та відповіді по заданих темах (на чому і натренована модель машинного навчання). Модель описана у модулі model.py, тренування моделі у train.py, файл data.pth - натренована збережена модель (зміна конфігурації файла .json потребує перетреновування моделі). 

Використано додатково завантажені бібліотеки: nltk, pytorch, numpy, uk_stemmer (https://github.com/Desklop/Uk_Stemmer)

Особливість програми полягає у можливості задання запитів боту у вільному форматі на основі нейромережі, натренованої на прикладах запитів, представлених у файлі .json (модель не є надто точною, тому для повноцінного тестування рекомендовано звірятися з тими ж прикладами запитів).

Для додавання нових функцій (в межах поданих категорій: фізика, математика, лінгвістика, географія, робота з текстом, загальне, інші проєкти) необхідно:
1) для простих текстових відповідей без маніпуляцій над вхідними даними: додати відповідний теґ, запит та відповідь у .json файлі за зразком. Перетренувати модель (запустити train.py)
2) для функцій з виконанням певних задач: зробити те саме + прописати відповідний клас з алгоритмом дій у модулі bot_classes.py
